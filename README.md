课程介绍：

视觉传达设计是为现代商业服务的艺术，主要包括标志设计、广告设计、包装设计、店内外环境设计、企业形象设计等方面，由于这些设计都是通过视觉形象传达给消费者的，因此称为“视觉传达设计”，它起着沟通企业——商品——消费者桥梁的作用。
